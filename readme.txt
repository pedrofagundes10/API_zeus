## Objetivos

Ativar o snmp do AP 310, e após mostrar o status da função.

## Requisitos

Possuir máquina com sistema operacional Linux (preferencialmente Ubuntu 20.04), com conexão local ao AP.
- Python 3
- Pip
- Biblioteca requests



## Instalação

### Python 3

Para instalar o Python 3, digite no terminal:

    $ sudo apt-get install python3

### Pip

Para instalar o Pip, o gerenciador de pacotes do Pyhton, digite no terminal:

	$ sudo apt-get install python3-pip

### Biblioteca requests 

Para instalar a biblioteca requests:

	$ pip3 install requests


## Execução

É recomendável que o status_snmp.py seja salvo em seu diretório de usuário (home/usuário).
Abra o diretório via terminal, e execute o software:

Para executar o software:

    $ python3 status_snmp.py

## Após executar será expresso o status do smnp.
